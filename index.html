Selected work

  1. 01

    MAY 2026 — PRESENT · BOARD BRING-UP

    Neural-network inference on a $15 FPGA

    Can a 9K-LUT hobby FPGA run real CNN inference with a fully open toolchain?

    An INT8-quantized MNIST accelerator written in Verilog for the GoWin GW1NR-9: parameterized multiply-accumulate datapath with INT32 accumulation, BRAM-backed weight storage, and a UART host interface. The whole flow is open source — Yosys synthesis, nextpnr place-and-route, USB-JTAG programming — running under Linux, with the accelerator now in board bring-up.

    INT8 MAC DATAPATH · FULLY OPEN TOOLCHAIN

  2. 02

    APR — JUN 2026 · V1 COMPLETE

    Inverse-designing silicon photonics with adjoint FDTD

    Hand-drawing nanophotonic geometry is guesswork — let gradients draw it instead.

    A Python topology-optimization pipeline on MEEP's FDTD adjoint solver, with every gradient verified against brute-force finite differences to under 1% before each run. It produced a 1×2 power splitter at 99.9% transmission (50.0/50.0 split) and a 1310/1550 nm wavelength duplexer near −17 dB crosstalk — re-optimized through a 0.12 µm minimum-feature filter so the final binary silicon/oxide geometry is actually fabricable.

    99.9% TRANSMISSION · 50.0/50.0 SPLIT

    • PYTHON
    • MEEP FDTD
    • NLOPT
    • NUMPY
  3. 03

    2025 — 2026 · DEPLOYED PROTOTYPE

    Recycla — waste sorting, camera to servo

    Recycling fails at the bin, where people guess. A camera doesn't have to.

    An end-to-end AI sorting system: a MobileNetV2 transfer-learning model trained across 5 material classes (~5,000 images) in TensorFlow/Keras, integrated with servo-actuated Raspberry Pi hardware that physically routes each item, plus a public web platform with real-time classification feedback. Trained the model, built the model-to-hardware integration, shipped the site.

    5 CLASSES · ~5,000 IMAGES
